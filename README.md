# Food-ordering-website
A full-stack web application built with Spring Boot (Java), PostgreSQL, and plain HTML/CSS/JavaScript.

Features
Browse restaurant menu
Add items to cart
Place orders
Tech Stack
Backend: Java, Spring Boot, Spring Security
Database: PostgreSQL
Frontend: HTML, CSS, JavaScript
Tool: Postman for API testing
How to Run
Clone the repo
Update application.properties with PostgreSQL DB credentials
Run OnlineFoodOrderingSystemApplication.java
